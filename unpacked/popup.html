Instruction

  • The extension works in devtool environment, so it can read the resources information. You will need to restart your browser or re-open developer tool panel, and switch to the Resource Saver tab.
  • All resources will be saved in "Downloads / All Resources" folder with specific domain name folder.
  • By default, extension only downloads resources which have address matched with current tab domain, tick "Download from all domains ..." to download all resources from other domains also.
  • If "Compress all assets ... (Recommended)" is ticked, all resources will be added into one compressed file (all.zip).
